## Getting Started

### Installation

##### Linux

```shell
git clone https://github.com/merwin-asm/OpenCrawler.git
```
```shell
cd OpenCrawler
```
```shell
chmod +x install.sh && ./install.sh
```

##### Windows

*You need git, python3 and pip installed*

```shell
git clone https://github.com/merwin-asm/OpenCrawler.git
```
```shell
cd OpenCrawler
```
```shell
pip install -r requirements.txt
```
